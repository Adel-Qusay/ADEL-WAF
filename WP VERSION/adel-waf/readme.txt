ADEL WAF is a lightweight WordPress Web Application Firewall.

== Description ==

ADEL WAF is a lightweight WordPress Web Application Firewall. Protects against: 
    Cross-site scripting (XSS)
    SQL injection (SQLI)
    Remote file inclusion (RFI)
    Remote code execution (RCE)
    Local file inclusion (LFI)
    Denial of service (DOS)
    Web shells
<strong>Email notification and logs can be activated and configured in <strong>Settings WP ADEL WAF</strong>. Go to your WP ADEL WAF settings page.
